System Architecture

Factory Inventory Management System

A full-stack demo for tracking factory inventory, orders, demand, backlog, and spending across warehouses. A Vue 3 single-page app talks to a Python FastAPI backend that serves filtered views over in-memory mock data — no database.

Frontend · Vue 3 + Vite · :3000 Backend · FastAPI + Uvicorn · :8001 Data · JSON in memory Transport · REST / JSON over HTTP

Data flow — a filtered request

How a user changing a filter turns into rendered data. The global filter bar drives every data view.

User adjusts the filter bar
FilterBar.vue is bound to the useFilters singleton: selectedPeriod, selectedLocation, selectedCategory, selectedStatus.
View reads current filters
getCurrentFilters() maps UI names to API params — selectedPeriod → month, selectedLocation → warehouse.
api.js issues an HTTP request
Axios builds a query string (skipping 'all') and GETs http://localhost:8001/api/…. The base URL is hardcoded in api.js.
FastAPI handles the route
The matching handler in main.py reads query params and filters the in-memory lists from mock_data.py.
Pydantic validates & serializes
List endpoints use response_model (e.g. List[Order]); dashboard, spending, and reports return computed dicts.
SPA renders the response
JSON lands in Vue refs; computed properties derive totals and chart series; SVG charts and tables update reactively.

Architectural notes & conventions

Non-obvious decisions worth knowing before changing the system.

Filter name indirection
UI refs selectedPeriod / selectedLocation are renamed to API params month / warehouse in getCurrentFilters(). Inventory has no time dimension, so it ignores month.
Stateless, in-memory backend
Data is read from JSON into module globals once at import. There is no database and no write path — restarting the server resets everything.
Hardcoded API base URL
API_BASE_URL is fixed to http://localhost:8001/api in api.js — there is no environment-variable override.
Canonical filter values
Warehouses: San Francisco, London, Tokyo. Categories: Actuators, Circuit Boards, Controllers, Power Supplies, Sensors. Statuses: Delivered, Shipped, Processing, Backordered.
Client calls without a backend route
api.js also defines /api/tasks* and /api/purchase-orders* calls, but main.py implements neither — those requests currently 404.
Open CORS
The backend allows all origins (allow_origins=["*"]) — appropriate for a local demo, not for production.
